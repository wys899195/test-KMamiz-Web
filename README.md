# KMamiz-Web

> This project is the frontend webpage for [KMamiz](https://github.com/stanley2058/KMamiz).

## Build Image

To build the image, run:

```bash
./build.sh
```

## Build Locally

First, install all dependencies by running `npm i`.

### To run the dev server:

```bash
npm run dev
```

### Build using Vite:

```bash
npm run build
```
